# ValuableSpawnConfig

Configure valuable spawn amounts per tier and rarity. Global and per-tier multipliers.

R.E.P.O. BepInEx mod. Host-side only.

## Install

Requires [BepInEx 5.x](https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/).

Place `ValuableSpawnConfig.dll` in `BepInEx/plugins/`.

## Configuration

Edit `BepInEx/config/maxenterme.ValuableSpawnConfig.cfg` after first launch.

## Build

```
dotnet build -c Release
```
